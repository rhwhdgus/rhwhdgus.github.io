# 기초-비트시프트연산

1047번
```cpp
#include <stdio.h>

int main()
{
	long long int a;
	scanf("%lld",&a);
	printf("%lld",a<<1);
	return 0;
}
```

1048번
```cpp
#include <stdio.h>

int main()
{
	int a,b;
	scanf("%d %d",&a,&b);
	printf("%d",a<<b);
	return 0;
}
```
